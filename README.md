# OSV2020 apportionment wrapper

Prerequisites: Java Development Kit (JDK), tested with OpenJKDK 25

First download the OSV2020 source code:
```
./download-osv2020-code.sh
```
Then install the apportionment wrapper:
```
./gradlew installDist
```
